# 🎓 Grade Calculator (Python)

ระบบคำนวณเกรดเฉลี่ยจากรายการคะแนน (List of Scores) โดยใช้ภาษา Python โปรแกรมนี้ถูกออกแบบมาให้มีความแม่นยำและรองรับกรณีข้อผิดพลาดเบื้องต้น (Edge Cases)

## 📝 คุณสมบัติ (Features)
- คำนวณคะแนนเฉลี่ยจาก `list` ของตัวเลข
- ตัดเกรดอัตโนมัติตามช่วงคะแนน (A, B, C, D, F)
- **Robustness:** รองรับกรณีส่งลิสต์ว่าง (`[]`) เข้ามา โดยไม่ทำให้โปรแกรมพัง (ZeroDivisionError)
- **Data Validation:** มีการตรวจสอบเบื้องต้นว่าข้อมูลต้องเป็นตัวเลข
- **Pythonic Code:** ใช้ฟังก์ชัน `sum()` เพื่อประสิทธิภาพการทำงานที่รวดเร็ว

## 🚀 วิธีการใช้งาน (Usage)

1. คัดลอกฟังก์ชัน `calculate_grade` ไปวางในโปรเจกต์ของคุณ
2. เรียกใช้งานฟังก์ชันโดยส่ง `list` ของคะแนนเข้าไป

### ตัวอย่างโค้ด:
```python
from grade_calculator import calculate_grade

# ตัวอย่างข้อมูลคะแนน
scores = [85, 92, 78, 88, 95]

# เรียกใช้งาน
grade, average = calculate_grade(scores)

print(f"Average: {average}") # ผลลัพธ์: 87.6
print(f"Grade: {grade}")     # ผลลัพธ์: A
```

## 🛠 การแก้ไขจากเวอร์ชันก่อนหน้า (Bug Fixes)

เวอร์ชันนี้ได้รับการปรับปรุงจากโค้ดเริ่มต้นเพื่อแก้ไขปัญหาดังนี้:
1. **ป้องกัน ZeroDivisionError:** เพิ่มการตรวจสอบ `if not scores` เพื่อป้องกันโปรแกรมหยุดทำงานเมื่อไม่มีการส่งคะแนนเข้าฟังก์ชัน
2. **ปรับปรุงประสิทธิภาพ:** เปลี่ยนจากการใช้ `for loop` บวกสะสมค่า เป็นการใช้ `sum()` ซึ่งเป็นฟังก์ชัน Built-in ของ Python ที่ทำงานได้รวดเร็วกว่า
3. **ความถูกต้องของข้อมูล:** เพิ่มการจัดการกรณีข้อมูลที่ส่งมาไม่ใช่ตัวเลข
